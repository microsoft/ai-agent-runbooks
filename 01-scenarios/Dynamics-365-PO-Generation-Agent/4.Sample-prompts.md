> These sample prompts are designed to help you get started with using the PO generator agent

---

## How to Use These Prompts

- Copy any prompt below and paste it directly into the **PO Generator Agent** chat
- You can use these prompts in:
  - **Microsoft Teams** — via the agent added under *Built for your org*
  - **M365 Copilot** — via the agent panel on the right navigation
  - **Email** — send an email to the inbox attached to the trigger with an attached PDF PO request, make sure "Purchase" is in the subject line
- Feel free to modify the prompts to match your specific situation

---

## Category 1: Using the agent directly to make a query

These prompts that procurment teams can ask the agent to help them more effeciently make reports on past POs, get vendor information, and understand what is going on in their current F&O environment.

| # | Prompt 
|---|---|
| 1 |  Create a summary report of PO to invoice cycle time by vendor, exportable as CSV. |
| 2 | Validate the attached CSV mapping for vendor master import and list likely import errors.|
| 3 | Recommend steps to deduplicate vendor master records and preserve transaction history.|
| 4 | Show the audit trail for changes to vendor X over the last 90 days. |
| 5 | Rank suppliers by on‑time delivery and quality metrics over the last 3 months. |
