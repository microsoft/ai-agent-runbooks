# Sample Prompts — Workplace Agent

These sample prompts help employees, inquiry administrators, and delivery engineers validate the **Workplace Agent**. Prompts are organized around the routing behavior implemented in the agent:

- Internal inquiries use the SharePoint Public FAQ only.
- Unresolved internal inquiries become Dataverse inquiry tickets.
- External Microsoft-product inquiries use Microsoft Learn Docs MCP.
- External non-Microsoft inquiries use Web search.
- Microsoft 365 Copilot license requests use the License Request Flow.

---

## How to Use These Prompts

Copy any prompt below and paste it into the **Workplace Agent** chat.

You can use these prompts in:

- **Copilot Studio Test Chat** — for maker and delivery validation.
- **Microsoft Teams** — through a direct chat with the published agent.
- **Microsoft 365 Copilot** — if the agent is published to Microsoft 365 Copilot.

> 💡 **Tip:** Add response-format instructions such as "show this as a table", "summarize in bullet points", or "give me step-by-step instructions" when testing response formatting.

## Category 1: Internal HR Inquiries

These prompts should be classified as **internal inquiries** and answered only from the SharePoint Public FAQ.

| # | Prompt | Expected Output |
|---|--------|-----------------|
| 1 | I'd like to learn about the childcare leave and family care leave programs. | FAQ-grounded answer describing that details are in company work regulations and HR guidelines, with guidance to contact HR for procedures and documents. |
| 2 | How can I apply for an internal transfer? | FAQ-grounded answer describing the internal job posting program, bulletin board / HR announcements, and HR career consultations. |
| 3 | I received a notice for a health check-up follow-up examination. What should I do? | FAQ-grounded answer instructing prompt medical follow-up and reporting results to HR. |
| 4 | Is there a consultation desk for employees? | FAQ-grounded answer describing harassment and mental-health consultation desks handled by HR. |
| 5 | Is having a side job permitted? | FAQ-grounded answer stating that side jobs require prior consultation and company approval. |

### Example Interaction

> **User:** Is having a side job permitted?
>
> **Agent:** Rules regarding side jobs are set out in the company work regulations. As a general rule, side jobs may not be undertaken without the company's approval. If you are considering a side job, please consult with the HR Department in advance and obtain the company's approval.
>
> Did this resolve your question?

These prompts should be classified as **external** and answered using **Microsoft Learn Docs MCP**.

| # | Prompt | Expected Output |
|---|--------|-----------------|
| 21 | What can Copilot in Excel do? | Microsoft Learn-grounded answer summarizing Copilot in Excel capabilities. |

| 25 | What is Power Automate and how does it integrate with Dataverse? | Microsoft Learn-grounded answer describing flow automation and connectors. |

### Example Interaction

> **User:** What can Copilot in Excel do?
>
> **Agent:** Copilot in Excel can help analyze and summarize data, generate formulas, highlight insights, create charts, and support natural-language data exploration. It works best when data is formatted in a table and the workbook is stored in a supported Microsoft 365 location.
>
> *(Source: Microsoft Learn Docs MCP.)*
